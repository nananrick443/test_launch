このゲームは、2000年代初頭に作られたゲームをHTML形式（Webページの形式）にて Scratch221171(Twitter/Qiita:@Scratch221171)が再現したものになります。
もともとのアプリは元データ\saitama.exeです。readmeも入っておりますので、ご活用ください。(DirectX3以上をインストールする必要があります)

~プレイ方法~
・タイトル画面で[Enter]キーを押すとゲームスタートです。
・[ESC]キーを押すとゲームを終了します。
・[F4]キーを押すとフルスクリーンになります。(フルスクリーンでやる必要ある？)

ゲームがスタートすると、
「さ」「い」「た」「ま」の４文字がバラバラの順番で横一列に表示されます。
例えば、「い ま さ た」とか「た い ま さ」の様になります。
この４文字の表示位置が、キーボードの[Z][X][C][V]キーに対応しています。
「い ま さ た」の場合、
「い」が[Z]、「ま」が[X]、「さ」が[C]、「た」が[V]です。
このバラバラの順番の４文字を、「さ い た ま」の順になるように対応するキーを押していくゲームです。
「い ま さ た」の場合、[C]→[Z]→[V]→[X]の順で押せば「さ い た ま」になります。
これをただひたすら繰り返します。

~起動しない場合~
スプラッシュ画面（さいたまモナー達とさいたま太陽の画像）のまま進まないとき、[Enter]キーを押すと上手くいくことがあります。
それでもうまくいかない場合は、「元データ」フォルダ内のsaitama.exeでプレイ可能です。
MacやLinuxではWine CrossOverを導入することでWindowsアプリを実行できます。

  あんまりキャラが活かせてない気もします。
  ノリはアーケードゲームの「ビシバシチャンプ」のそれに近いかもしれません。
  気楽に作ったので気楽に遊んで下さい。


[2]. ____動作環境____
  OSは Windows 95/98/2000 で。
  Meは微妙。XPは手元のXPで動作してるので多分OK。
  DirectX 3以上がインストールされてる事（Win95以外は初期状態でインストール済です）。
  それなりにイイ感じの速度のCPUを搭載してる事（いいかげん）。
  基本的にロースペックのマシンでも動作するかと思われます。
